Taxi Fare Prediction – Task 2 (Machine Learning Model)

This repository contains my Task 2 submission for the FutureIntern program.
In this task, I built a Taxi Fare Prediction Machine Learning Model using Python and common data-science libraries.

🚖 Project Overview

The goal of this project is to predict taxi fares based on features like:

Distance traveled
Passenger count
Pickup & drop-off coordinates
Date & time of travel

The model is trained using a linear regression algorithm.

📌 Features
Data loading and preprocessing
Splitting dataset into train & test sets
Training a regression model
Predicting taxi fare
Model evaluation using:
R² Score
Mean Squared Error (MSE)
